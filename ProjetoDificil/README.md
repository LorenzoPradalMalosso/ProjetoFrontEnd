## Calculadora Simples

Este diretório contém um pequeno projeto front-end que implementa uma calculadora simples em HTML, CSS e JavaScript puro.

## Estrutura do Projeto

- `ProjetoDificil/`
  - `index.html` – marcação da calculadora e referência aos arquivos de estilo e script.
  - `style.css` – folhas de estilo para layout e aparência.
  - `script.js` – lógica JavaScript responsável por capturar cliques, construir a expressão e calcular o resultado.

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão.
- Visor que exibe a expressão atualizada e o resultado.
- Botão "C" para limpar a expressão.
- Tratamento simples de erros de sintaxe na expressão.

## Como usar

1. Abra `ProjetoDificil/index.html` em um navegador.
2. Clique nos botões numéricos e de operação para construir uma expressão matemática.
3. Pressione `=` para calcular o resultado ou `C` para limpar.

## Tecnologias

- HTML
- CSS
- JavaScript

## Observações

Este projeto é educativo e não faz validação avançada de entrada — o `eval` é usado para avaliar expressões e assume-se que o usuário não insira código malicioso.
